# Серверное хранилище «сайт Автопилота»

Рабочий каталог на Beget: `/opt/beget/сайт Автопилота`.

- `current/` — текущая сборка сайта и медиафайлы.
- `data/` — закрытое хранилище IQ Автопилота.
- `graph-archive/` — снимки проекта и карта его структуры.
- `backups/` — архивные копии сборок.

Текущий счёт IQ в браузере сохраняется под прежним ключом, чтобы не обнулить уже набранные баллы. Серверная база готова под будущую авторизацию, синхронизацию баланса и обмен баллов на награды. База не публикуется в веб-корне.
