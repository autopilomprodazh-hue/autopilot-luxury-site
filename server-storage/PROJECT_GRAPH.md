# Карта проекта «Автопилот продаж»

- `t123-fragment.html` — главная сборка; подключает медиафайлы из `assets/`.
- `assets/` — 3D-сцены и иллюстрации страницы.
- `localStorage: aps_luxury_iq_v1` — текущий локальный счёт IQ; ключ сохранён для совместимости.
- `data/iq-autopilot.sqlite` — закрытый серверный журнал начислений и обмена IQ.
- `server-storage/iq-schema.sql` → `data/iq-autopilot.sqlite` — схема базы.
- Будущий авторизованный API свяжет браузерный счёт с серверным журналом; до авторизации публичная запись в базу отключена.
- Tilda: дубликат страницы `154402226`; обновление требует входа в аккаунт проекта `12700657`.

Архивные снимки хранятся в двух контурах:

1. `/opt/beget/сайт Автопилота/graph-archive/`.
2. `/opt/beget/graph-archive-autopilot/site-autopilot/`.
